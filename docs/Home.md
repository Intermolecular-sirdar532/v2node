# v2node Wiki

欢迎来到 v2node 的官方 Wiki！

## 项目简介

v2node 是一个基于修改版 xray-core 的 V2board 节点服务端，支持多种协议（VLESS、VMESS、Trojan、Shadowsocks、Hysteria2、TUIC、AnyTLS）。
